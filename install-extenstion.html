How to Install and Use VS Code's Live Server Extension

Install | Use

Install Instructions

  1. In visual studio code, click the extensions icon in the left column toobar.
  2. In the search bar that appears, type Live server
  3. Select the live server extension. It should be the top choice.
  4. In the page for live server that appears, click the green install button.

Usage Instructions

  • Visual studio code mustbe open to a folder to use Live Server.
  • In VS code's explorer, right click (windows) or ctrl + click (macOS) on an HTML file. From the context menu that appears, select open with live server.
